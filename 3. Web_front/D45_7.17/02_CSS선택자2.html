속성 선택자

기본 선택자 뒤에 []를 사용하여 속성과 속성값을 사용한다.

선택자[속성=속성값] {속성:속성값;}의 형식으로 사용한다.

동위 선택자

동위 관계에서 뒤에 위치한 태그를 선택할 때 사용한다.

        바로 뒤에 있는 요소를 선택 시 : 선택자a + 선택자b {설정내용}
        뒤에 있는 모든 요소 선택 : 선택자a ~ 선택자b {설정내용}

반응선택자

사용자의 움직임에 따라 달라지는 선택자

        사용자가 클릭한 태그 : 선택자:active {설정내용}; 
        사용자의 마우스가 올라가 있는 태그 : 선택자:hover {설정내용};으로 사용된다.

상태선택자

입력양식의 상태에 따라 선택되는 선택자

체크 상태의 input 태그 선택 : input:checked 

초점이 맞추어진 input태그 선택


        사용 가능한 input태그 : input태그:enable {설정내용}
        사용 불가능한 input태그 : input태그:disabled {설정내용}
