텍스트 스타일

color 속성 : 텍스트에 색상을 지정할 수 있다.

선택자 {color : 색상명영문이름 | 16진수숫자 | rgb값 | rgba값 | hsl값 | hsla값;}의 형식으로 사용된다.
  • 색상명으로 지정
  • 16진수 값으로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla 지정

text-decoration 속성 : 텍스트에 줄을 긋거나 줄을 없앨 때 사용한다.

선택자 {text-decoration: none | underline | overline | line-through;}의 형식으로 사용됨
  • 하이퍼링크 밑줄 없애기
  • 텍스트 영역 아래 줄 긋기
  • 텍스트 영역 중간에 줄 긋기
  • 텍스트 영역 위에 줄 긋기

text-shadow 속성 : 텍스트에 그림자 효과를 줄 때 사용한다.

선택자 : {text-shadow : none | 가로거리 세로거리 번짐정도 색상;}
