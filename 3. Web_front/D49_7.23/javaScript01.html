자바스크립트 개요

자바스크립트
웹 브라우저에서 많이 사용하는 인터프리터 방식의 객체 지향 프로그래밍 언어이다.
자바스크립트 사용법
  1. 태그에 직접 간단한 소스코드를 작성해서 실행되게 하는 방법(인라인)
  2. script 태그 영역에 작성해서 실행되게 하는 방법
  3. 별도의 js 파일로 작성해 가져다 사용하는 방법
단, 소스코드는 함수 단위로 작성하고 이벤트 속성을 사용해서 구동되게 한다.
이벤트 속성은 태그마다 차이가 있음
on이벤트명="실행할함수명([전달값]);"

1. 태그에 직접 자바스크립트 코드를 작성해서 실행되게 하는 경우


2. script 태그 안에 함수 단위로 소스 작성하고, 태그에서 on이벤트명="함수명();"으로 사용하는 경우


3. 별도로 작성된 자바스크립트 함수 사용 테스트
