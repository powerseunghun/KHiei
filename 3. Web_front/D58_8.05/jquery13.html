객체 생성 및 제거 메소드

삽입메소드 part1

$(A).appendTo(B) : A를 B의 요소 내 뒷부분에 추가
$(B).prependTo(B) : A를 B의 요소 내 앞부분에 추가
$(A).insertAfter(B) : A를 B의 요소 뒤에 추가
$(A).insertBefore(b) : A를 B의 요소 앞에 추가

삽입메소드 part2

$(A).append(B) : B를 A요소 내 뒷부분에 추가
$(B).prepend(B) : B를 A요소 내 앞부분에 추가
$(A).after(B) : B를 A요소 뒤에 추가
$(A).before(b) : B를 A요소 앞에 추가

객체 복사와 제거 메소드

clone() : html 내부에 있는 요소를 복사하는 메소드
clone(true) : 이벤트까지 같이 복사
