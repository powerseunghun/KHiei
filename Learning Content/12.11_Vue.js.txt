1. Vue.js
- jquery 에서는 값을 복사하기 위해 "태그"에 이벤트를 설정
- jquery에서는 "태그"를 어떻게 제어할 것인지에 대한 고민이 주 관심사
- VueJS는 두 가지 방법으로 사용 가능
  - 1. 라이브러리처럼 불러서 사용(단일 페이지 내에서 사용 가능)
  - 2. Webpack 등의 관리 프레임워크를 이용해서 프로젝트화 시켜서 사용이 가능
- 템플릿 제작
- 디렉티브
- 동적인 입력창 생성이라고 한다면.. jquery는 태그 기준이니까 어렵지만 vue는 데이터 기준이므로 훨씬 쉬워짐
- v-model : Vue의 데이터와 입력창을 "양방향" 연결할 때 사용
- v-bind : 동적으로 속성을 부여하기 위한 조건부 속성 설정 코드
- v-if : 조건을 충족할 때 태그를 생성하는 코드
- v-show : 조건을 충족할 때 display 속성을 변경하여 표시하는 코드
- v-on : 이벤트를 설정하기 위한 코드이며 콜론(:) 뒤에 이벤트의 종류를 표시(종류는 javascript와 동일)
- {{식}} : 표현식이라 부르며 Vue의 데이터를 출력할 때 사용
